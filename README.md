All of the commands except `eject` will still work, but they will point to the copied scripts so you can tweak them. At this point you're on your own.

You don't have to ever use `eject`. The curated feature set is suitable for small and middle deployments, and you shouldn't feel obligated to use this feature. However we understand that this tool wouldn't be useful if you couldn't customize it when you are ready for it.

# Titulo y Descripción

Calculadora y ToDoList

Realizamos una calculadora y una lista de tareas con la librería React con la ayuda de freeCodeCamp.

La Calculadora:

El usuario podrá un ingresar un número el display haciendo click en el teclado que aparece en la calculadora.

El usuario puede realizar la 4 operaciones como sumar, restar, multiplicar y dividir.

El usuario podrá ver el resultado de la operación en el display de la calculadora.

El usuario no puede utilizar el teclado de la pc para ingresar valores en el display de la calculadora.

El ToDoList:

El usuario podrá ingresar una tarea y al hacer click en el boton 'agregar tarea' se mostrará en la parte inferior del input.

El usuario podrá ingresar la cantidad de tareas que desea y estas aparecerán en un listado visual.

El usuario tendrá la opción de tachar y destachar la tarea.

El usuario tendrá la opción de eliminar la tarea.


## Instalaciones

Para evaluar las operaciones usamos la librería math:

`npm install mathjs`

Para generar un Id utilizamos la librería uuid:

`npm i uuidjs`

Para los íconos usamos la librería react-icons:

`npm i react-icons`
